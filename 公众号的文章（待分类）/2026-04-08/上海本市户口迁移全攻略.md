01  

适用场景（你最关心的两种）  

✅ **本市户籍→本人 / 配偶 / 父母产权房（直系）**：最常见，为了对口入学、人户一致，小升初第一顺位关键；

✅ **本市户籍→社区公共户 / 非直系亲属户**：人户分离、租房、无产权房家庭适配。

  

---

02  

完整办理流程

（全市通办，不用跑户籍地）  

### Step1 准备材料（核心必带，缺一不可）

1）**本人 / 直系产权房迁入（最稳、入学用）**

*   迁移人身份证、原户口本 / 个人户口卡（原件）
    
*   入户地房产证 / 不动产登记证明（原件，父母 / 配偶产权需提供亲属关系证明：出生证、结婚证）
    
*   入户地户口本（原件，产权人 / 户主同意入户签字）
    
*   迁移申请表（派出所现场填）
    

2）**迁入社区公共户（人户分离、租房）**

*   身份证、原户口本
    
*   房屋租赁备案（网签 + 备案，有效期内）
    
*   实际居住承诺、无房证明（不动产登记中心开）
    
*   社区公共户入户申请表
    

3）**非直系亲属 / 挂靠（不推荐，入学易被统筹）**

同上 + 非直系亲属同意入户意见书、亲属关系证明

  

### Step2 提交申请（全市通办，就近派出所）

选任意派出所综合窗口（不用去原户籍地、迁入地），现场取号、交材料、填申请表；民警核验材料、拍照、录入系统，当场告知是否受理、是否缺材料。

###   

### Step3 审核 + 办结（分 2 种情况）

情况 A：**直系 / 本人产权房（当场办结）**

材料齐全、无异常（无冻结、无空挂、无纠纷）→当场审核通过、打印新户口本 / 更新内页、换户口卡，立等可取（10-30 分钟）

情况 B：**社区公共户 / 非直系 / 跨区复杂**

非户籍地派出所受理→3 个工作日内（迁入地派出所审核）→审核通过后，通知去窗口打印新户口本 / 户口卡

特殊情况（产权纠纷、冻结户、材料存疑）：延长至5 个工作日，以派出所通知为准

  

### Step4 后续（入学关键）

拿到新户口本后，立即核对地址、户主、产权人信息（错了当场改）；小升初报名前，确保人户一致（户籍 = 实际居住 + 产权），并保留户口本、房产证、居住证明原件。

03  

办理时间

（精确到工作日，不含周末 / 法定假日）  

1）**最快**：直系 / 本人房产（当场办结）

办理时长：1 个工作日内（当场），材料齐 10-30 分钟拿证；总周期：0-1 天（当天去、当天好）

2）**常规**：社区公共户 / 非直系

受理→审核→办结：3 个工作日（不含提交当天）；总周期：3-5 天（提交日 + 3 个工作日）

3）**特殊延迟（避坑）**

材料不全、缺签字、产权有抵押 / 查封、户口冻结、空挂户：补材料后重新算时间，可能 7-10 天；中心城区（徐汇、静安、黄浦等）热门片区户籍审核更严，偶尔会延长 1-2 天核验。

入学 / 人户分离关键提醒

✅ **入户时间节点（小升初生死线）**：上海小升初一般4 月 30 日截止户籍信息核验，务必提前 1-2 个月办好户口迁移，留足材料补正、审核时间，避免卡节点；

二、入户核查全清单（入学必核，逐项打勾）  

### （一）基础户籍核查（派出所必查）

✅ 入户地址有效性：房屋真实存在、无灭失 / 拆迁 / 违建、非虚拟地址，派出所系统可落户口；

✅ 户口冻结排查：该地址是否被户籍冻结（热门片区入学季常冻结）、是否有户口纠纷、是否有抵押 / 查封限制入户；

✅ 产权 / 承租权核验：

本人 / 父母 / 配偶产权：房产证 / 不动产证原件、产权人全部同意入户签字（共有产权需所有产权人同意）；

祖辈产权：需提供亲属关系证明（出生证）、实际居住承诺 + 同住证明，部分热门区要求祖辈产权 + 父母无房；

公房：承租卡原件、承租人同意入户；

✅ 迁移人材料：身份证、原户口本 / 户口卡、迁移申请表（现场填）、亲属关系证明（直系迁入必备）。

  

### （二）入学核心核查（决定是否对口、防统筹）

1. **五年一户核查（最关键，直接影响对口）**

核查目标：该房产5 年内（从入学当年倒推）是否有孩子占用对口公办学位（多胎除外）；

核查渠道：

*   派出所户籍窗口：报地址查学位占用记录；
    
*   随申办 APP→入学报名→学区查询→五年一户核验；
    
*   对口学校教务处 / 区教育局招生办二次确认；
    

结论：已占用→即便人户一致也会被统筹；未占用→符合年限才稳。

2. **人户一致核查（第一顺位标准）**

户籍地址 = 房产证地址（完全一致，门牌号无差异）；

户主：本人 / 父母 / 配偶（直系），非挂靠、非集体户、非公共户；

产权：父母 / 本人全产权（优先），祖辈产权需同住 + 父母无房；

实际居住：派出所 / 社区上门核验（热门片区必查），空挂户 = 人户不一致，直接统筹。

3. **入户年限核查（热门校硬门槛）**

徐汇：学籍满 3 年 + 人户一致满 3 年（截止 4 月 30 日）；

静安：人户一致满 3 年（静教院、市西、市北）；

黄浦 / 虹口 / 杨浦 / 普陀 / 长宁：人户一致满 2-3 年，超员按入户时间排序；

人户分离 / 公共户：无年限优先级，直接统筹。

  

### （三）风险排除清单（避坑）

✅ 该地址无户口冻结、无产权纠纷、无查封抵押

✅ 五年一户未占用（随申办 + 派出所双查）

✅ 产权人 / 户主全部同意入户、签字齐全

✅ 迁移材料（身份证、户口本、房产证、亲属证明）原件齐全

✅ 实际居住（非空挂），能提供水电煤账单 / 居委会证明

✅ 入户时间在小升初 4 月 30 日截止前，预留补材料时间
